更新 ======================

a：增加了抽奖前的判断：

1.当天是否已抽过奖， ‘否’的话进入下一条判断 
（这里用localstorage储存，获取当天24点时间戳，过了当天24点localstorage储存清除，时间都是获取本地时间，实际应用应采用服务器时间，防止用户修改本地时间来作弊）


2.现在抽奖时间是否在16-17点之间，‘是’的话则开始抽奖
